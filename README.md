# Hear-Disease-ML-prediction
This code trains Machine Learning algorithms to predict heart diseases of medical patients. Supervised Learning binary classification task. DataSet available at the root directory: `data.csv`.

Language: Python

Libraries: 
* pandas
* numpy
* matplotlib
* sklearn

Models:
* Decision Tree
* Random Forest
* kNN
* Logistic Regression
* AdaBoost with Decision Trees as weak estimators

Model Testing Accuracy requested > **80 %** - successfully reached with each model. Highest Accuracy: kNN and Logistic Regression (87,4 %).

For each model, code also calculates ROC Curves, AUC Scores and F1 Scores.

Details included in the .ipybn jupyter file as markdowns.
